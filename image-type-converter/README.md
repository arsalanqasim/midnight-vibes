# 🖼️ Image Converter

A simple GUI tool to convert images between different formats. Perfect for converting iPhone photos (HEIC) to more common formats like PNG or JPEG.

## What it does

- Convert images between PNG, JPEG, GIF, BMP, TIFF, WEBP, and HEIC formats
- Batch convert multiple files at once
- Adjust quality settings
- Resize images while keeping proportions
- Preview images before converting

## How to use

1. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the app**:
   ```bash
   python image_converter_gui.py
   ```

3. **Convert images**:
   - Select files or folders
   - Choose output format
   - Pick where to save converted files
   - Click "Convert Files"

## Quick tips

- **iPhone photos**: Great for converting HEIC files to PNG/JPEG
- **Batch processing**: Select entire folders to convert all images at once
- **Quality control**: Use the slider to adjust output quality
- **Resizing**: Enable resizing to make images smaller

That's it! Simple and straightforward. 🌙

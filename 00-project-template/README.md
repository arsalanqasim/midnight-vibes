# Project Template

This is a template for new projects in the midnight-vibes repository.

## 📁 Project Structure
```
XX-project-name/
├── main.py                 # Main application file
├── requirements.txt         # Dependencies
├── README.md               # Project-specific documentation
├── LICENSE                 # Project license (if different from repo)
├── .gitignore             # Project-specific gitignore (if needed)
└── assets/                 # Images, data, etc. (if needed)
```

## 🚀 Getting Started

1. **Copy this template** to a new numbered folder
2. **Rename** the folder to `XX-project-name` (where XX is the next number)
3. **Update** the README.md with project-specific information
4. **Add** your code and dependencies
5. **Update** the main repository README.md to include your project

## 📝 README Template

```markdown
# Project Name

Brief description of what this project does.

## ✨ Features
- Feature 1
- Feature 2
- Feature 3

## 🚀 Installation
```bash
pip install -r requirements.txt
```

## 📖 Usage
```bash
python main.py
```

## 🛠️ Tech Stack
- Python
- Other dependencies

## 📄 License
MIT License
```

## 🎯 Project Naming Convention

- Use sequential numbers: `01-`, `02-`, `03-`, etc.
- Use descriptive names: `image-converter-gui`, `web-scraper-tool`, etc.
- Use kebab-case: `my-awesome-project`
- Keep it short but descriptive

## 🌙 Midnight Vibes Guidelines

- **Keep it simple** - Don't overcomplicate
- **Make it useful** - Solve real problems
- **Document well** - Future you will thank you
- **Have fun** - This is your creative space

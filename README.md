## 🚀 Projects

### 01. 🖼️ [Image Converter GUI](./01-image-converter-gui/)
**Advanced Image File Converter with Professional GUI**

A feature-rich image conversion tool that handles multiple formats with batch processing, quality control, resizing, and more. Perfect for photographers, web developers, and anyone who works with images.

**Tech Stack:** Python, tkinter, Pillow, pillow-heif  
**Features:** Multi-format support, batch processing, quality control, preview, history tracking
